# onlinebookstore
Project Overview
Book Learn is a static online bookstore website designed to showcase a wide range of books across multiple categories. The project focuses on clean layout design, organized content structure, and ease of navigation. It is suitable for academic coursework, front-end practice, and portfolio demonstration.
Objectives

To design a structured and user-friendly bookstore interface

To organize books into meaningful categories

To practice HTML, CSS, and basic JavaScript integration

To manage assets (images) using a proper folder hierarchy
