# Online Bookstore

## Project Overview
Book Learn is a static online bookstore website designed to provide users with a visually appealing and organized platform to explore and browse books across multiple categories. The project emphasizes simplicity, usability, and responsiveness to ensure a smooth user experience.

## Objectives
- Design a structured and user-friendly bookstore interface.
- Organize books into meaningful categories for easy navigation.
- Provide detailed information for each book including title, author, and description.
- Offer a platform suitable for front-end practice and portfolio demonstration.

## Features
### Book Categories
- **Fiction:** Novels, short stories, and literary works.
- **Non-Fiction:** Biographies, self-help, and educational books.
- **Sales Collection:** Books currently on sale or discounted.
- **Children Books:** Books aimed at children of various age groups.
- **New Arrivals:** Latest books recently added to the collection.
- **Urdu Books:** Books available in the Urdu language.
- **Academic Books:** Textbooks and reference books for students.

### Additional Features
- Responsive design for desktop, tablet, and mobile devices.
- Search bar to find books by title or author.
- Clear navigation through categories and sections.
- Well-structured layout for readability and aesthetic appeal.

## Technologies Used
- **HTML5** – for website structure.
- **CSS3** – for styling and layout design.
- **JavaScript** – for interactive elements like search functionality.
- **Git & GitHub** – for version control and deployment.
